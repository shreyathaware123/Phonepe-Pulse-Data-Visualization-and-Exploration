##  Project Information:

This project focuses on **visualizing and exploring PhonePe Pulse data** to uncover insights into digital payment trends across India. The data is transformed and presented through an **interactive geo visualization dashboard** that allows users to explore transactions, users, growth rates, and regional patterns in a user-friendly manner.

## Problem Statement:

The Phonepe pulse Github repository contains a large amount of data related to various metrics and statistics. The goal is process the data to obtain insight and information that can be visualized in a user-friendly manner.

## ğŸ“‚ Dataset

Aggregated_Transaction.csv - 
Contains aggregated transaction metrics such as transaction count and total transaction value across states and time periods.

Aggregated_User.csv - 
Includes aggregated user data such as registered users and app usage metrics by state and time.

Map_Transaction.csv - 
Provides district-level transaction data used for geo-visualization and spatial analysis.

Map_User.csv - 
Contains district-level user information for mapping user distribution and adoption trends.

Top_Transaction.csv - 
Identifies top-performing states and districts based on transaction volume and value.

Top_User.csv - 
Highlights top states and districts based on registered users and user growth.

The dashboard presents **India-wide geo-visual analytics** with deep interactivity.

* **KPIs:**
